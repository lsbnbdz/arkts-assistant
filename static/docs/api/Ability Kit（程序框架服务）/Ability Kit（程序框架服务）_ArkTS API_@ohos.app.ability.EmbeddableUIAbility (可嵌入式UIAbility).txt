EmbeddableUIAbility是为元服务提供可以嵌入式启动的UIAbility，继承自[UIAbility](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-ability-uiability)，同时具备跳转启动和嵌入式启动两种启动方式。各类Ability的继承关系详见[继承关系说明](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-ability-ability#ability%E7%9A%84%E7%BB%A7%E6%89%BF%E5%85%B3%E7%B3%BB%E8%AF%B4%E6%98%8E)。

 说明 本模块首批接口从API version 12 开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

  本模块接口仅可在Stage模型下使用。

## 导入模块
支持设备PhonePC/2in1TabletTVWearable收起自动换行深色代码主题复制
```
import { EmbeddableUIAbility } from '@kit.AbilityKit';
```
   
## EmbeddableUIAbility
支持设备PhonePC/2in1TabletTVWearable系统能力：SystemCapability.Ability.AbilityRuntime.Core

 元服务API：从API version 12 开始，该接口支持在元服务中使用。
