## 概述
PhonePC/2in1TabletTVWearable提供NativeImage功能，作为数据消费者，其中一种用法是将数据和OpenGL纹理对接，需在OpenGL环境下使用，另外一种用法是开发者自行获取buffer进行渲染处理。

 **起始版本：** 9
