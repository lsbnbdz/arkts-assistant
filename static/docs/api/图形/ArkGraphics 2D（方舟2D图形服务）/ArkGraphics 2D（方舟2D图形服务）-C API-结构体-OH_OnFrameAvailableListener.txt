## 概述
PhonePC/2in1TabletTVWearable一个OH_NativeImage的监听者，通过[OH_NativeImage_SetOnFrameAvailableListener](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-image-h#oh_nativeimage_setonframeavailablelistener)接口注册该监听结构体，当有buffer可获取时，将触发回调给用户。

 **起始版本：** 11

 **相关模块：** [OH_NativeImage](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-oh-nativeimage)

 **所在头文件：** [native_image.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-image-h)
