# [GSNode（3DGS渲染对象）](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/spatial-recon-gsnode)

## 目录
- [导入模块](#section13218152033713)
- [使用说明](#section16641355295)

GSNode类继承自[Node](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-scene-nodes#node)，代表一个3DGS（3D Gaussian Splatting）渲染对象，帮助开发者操作3DGS模型。

 **系统能力**：SystemCapability.Graphics.SpatialRender

 **起始版本：**6.0.1(21)

 
## 导入模块
PhonePC/2in1TabletTV收起自动换行深色代码主题复制
```
import { spatialRender } from '@kit.SpatialReconKit';
```
 
## 使用说明
PhonePC/2in1TabletTVGSNode类与[Node](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-scene-nodes#node)等价，使用方法请参见[Node](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-scene-nodes#node)。
