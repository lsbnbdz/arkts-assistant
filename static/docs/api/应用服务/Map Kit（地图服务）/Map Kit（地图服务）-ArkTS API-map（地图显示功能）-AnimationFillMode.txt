## AnimationFillMode
PhonePC/2in1TabletWearable动画执行完成后的状态。

 **模型约束：**此接口仅可在Stage模型下使用。

 **元服务API：**从版本4.1.0(11)开始，该接口支持在元服务中使用。

 **系统能力：**SystemCapability.Map.Core

 **起始版本：**4.1.0(11)

 展开
| 名称 | 值 | 说明 |
| --- | --- | --- |
| FORWARDS | 0 | 动画执行后保持在最后一帧。 |
| BACKWARDS | 1 | 动画执行后保持在第一帧。 |
