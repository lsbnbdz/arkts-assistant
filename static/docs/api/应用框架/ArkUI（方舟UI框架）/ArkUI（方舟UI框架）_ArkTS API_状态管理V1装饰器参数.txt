## ProvideOptions
          支持设备PhonePC/2in1TabletTVWearable[@Provide](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-provide-and-consume)参数，用于配置@Provide的支持重写，具体例子可见[@Provide支持allowOverride参数](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-provide-and-consume#provide%E6%94%AF%E6%8C%81allowoverride%E5%8F%82%E6%95%B0)。

     卡片能力： 从API version 11开始，该接口支持在ArkTS卡片中使用。

     元服务API： 从API version 11开始，该接口支持在元服务中使用。

     系统能力： SystemCapability.ArkUI.ArkUI.Full

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| allowOverride | string | 否 | 配置@Provide支持重写。默认不支持重写。 |
