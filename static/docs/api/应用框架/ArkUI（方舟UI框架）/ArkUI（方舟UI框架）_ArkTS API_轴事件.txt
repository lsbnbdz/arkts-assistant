轴事件指组件被鼠标滚轮滚动或触控板双指沿特定方向（轴）滑动时触发的事件。“轴”指二维坐标系中的方向，分为水平（X轴）和垂直（Y轴）。

    说明      从API Version 17开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。

## onAxisEvent
          支持设备PhonePC/2in1TabletTVWearableonAxisEvent(event: Callback<AxisEvent>): T

     鼠标滚轮滚动或触控板双指滑动时触发该回调。

     元服务API： 从API version 17开始，该接口支持在元服务中使用。

     系统能力： SystemCapability.ArkUI.ArkUI.Full

     参数：

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| event | Callback< AxisEvent > | 是 | 获得 AxisEvent 对象。 |

          返回值：

|  |
|  |
|  |
| T | 返回当前组件。 |

## AxisEvent
          支持设备PhonePC/2in1TabletTVWearable轴事件的对象说明，继承于[BaseEvent](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-gesture-customize-judge#baseevent8)。

     元服务API： 从API version 17开始，该接口支持在元服务中使用。

     系统能力： SystemCapability.ArkUI.ArkUI.Full

### 属性
          支持设备PhonePC/2in1TabletTVWearable系统能力： SystemCapability.ArkUI.ArkUI.Full

| 名称 | 类型 | 只读 | 可选 | 说明 |
| --- | --- | --- | --- | --- |
| action | AxisAction | 是 | 否 |  轴事件的动作类型。 元服务API： 从API version 17开始，该接口支持在元服务中使用。 |
| x | number | 是 | 否 |  鼠标光标相对于被点击元素左边缘的X坐标。 单位：vp 元服务API： 从API version 17开始，该接口支持在元服务中使用。 |
| y | number | 是 | 否 |  鼠标光标相对于被点击元素上边缘的Y坐标。 单位：vp 元服务API： 从API version 17开始，该接口支持在元服务中使用。 |
| windowX | number | 是 | 否 |  鼠标光标相对于当前窗口左上角的X坐标。 单位：vp 元服务API： 从API version 17开始，该接口支持在元服务中使用。 |
| windowY | number | 是 | 否 |  鼠标光标相对于当前窗口左上角的Y坐标。 单位：vp 元服务API： 从API version 17开始，该接口支持在元服务中使用。 |
| displayX | number | 是 | 否 |  鼠标光标相对于当前屏幕左上角的X坐标。 单位：vp 元服务API： 从API version 17开始，该接口支持在元服务中使用。 |
| displayY | number | 是 | 否 |  鼠标光标相对于当前屏幕左上角的Y坐标。 单位：vp 元服务API： 从API version 17开始，该接口支持在元服务中使用。 |
| scrollStep | number | 是 | 否 |  鼠标轴滚动步长配置。 说明： 仅支持鼠标滚轮，取值范围：[0~65535] 元服务API： 从API version 17开始，该接口支持在元服务中使用。 |
| propagation | Callback<void> | 是 | 否 |  激活事件冒泡。 元服务API： 从API version 17开始，该接口支持在元服务中使用。 |
| globalDisplayX 20+ | number | 否 | 是 |  鼠标光标相对于全局屏幕的左上角的X坐标。 单位：vp 取值范围：[0, +∞) 元服务API： 从API version 20开始，该接口支持在元服务中使用。 |
| globalDisplayY 20+ | number | 否 | 是 |  鼠标光标相对于全局屏幕的左上角的Y坐标。 单位：vp 取值范围：[0, +∞) 元服务API： 从API version 20开始，该接口支持在元服务中使用。 |
