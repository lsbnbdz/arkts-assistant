设备上报的手势事件。

 说明 - 本模块首批接口从API version 10开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

 
## 导入模块
PhonePC/2in1TabletTVWearable收起自动换行深色代码主题复制
```
import { Rotate, Pinch, ThreeFingersSwipe, FourFingersSwipe, ActionType } from '@kit.InputKit';
```
 
## Pinch
PhonePC/2in1TabletTVWearable捏合手势事件。

 **系统能力**：SystemCapability.MultimodalInput.Input.Core

 展开
| 名称 | 类型 | 只读 | 可选 | 说明 |
| --- | --- | --- | --- | --- |
| type | ActionType | 否 | 否 | 手势事件类型。如：手势开始、手势更新、手势结束等。 |
| scale | number | 否 | 否 | 捏合度，取值范围大于等于0。 |

## ActionType
PhonePC/2in1TabletTVWearable手势事件类型。

 **系统能力**：SystemCapability.MultimodalInput.Input.Core

 展开
| 名称 | 值 | 说明 |
| --- | --- | --- |
| CANCEL | 0 | 取消。 |
| BEGIN | 1 | 手势开始。 |
| UPDATE | 2 | 手势更新。 |
| END | 3 | 手势结束。 |
