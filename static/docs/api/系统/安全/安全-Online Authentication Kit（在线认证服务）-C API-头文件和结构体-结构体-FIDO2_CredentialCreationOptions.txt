# [FIDO2_CredentialCreationOptions](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/_f_i_d_o2___credential_creation_options)

## 概述
PhonePC/2in1Tablet凭据请求的选项。

 **起始版本：** 6.0.0(20)

 **相关模块：** [FIDO2](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/passkey)

 
## 汇总
PhonePC/2in1Tablet 
### 成员变量
 PhonePC/2in1Tablet展开
| |
| |
| |
| FIDO2_CredentialMediationRequirement mediation | 该操作是否需要用户参与。 |
| FIDO2_PublicKeyCredentialCreationOptions publicKey | publicKey凭证请求的选项。 |

 
## 结构体成员变量说明
PhonePC/2in1Tablet 
### mediation
PhonePC/2in1Tablet收起自动换行深色代码主题复制
```
FIDO2_CredentialMediationRequirement FIDO2_CredentialCreationOptions::mediation
```
 **描述**

 操作是否需要用户参与。

 
### publicKey
PhonePC/2in1Tablet收起自动换行深色代码主题复制
```
FIDO2_PublicKeyCredentialCreationOptions FIDO2_CredentialCreationOptions::publicKey
```
 **描述**

 publicKey凭证请求的选项。
