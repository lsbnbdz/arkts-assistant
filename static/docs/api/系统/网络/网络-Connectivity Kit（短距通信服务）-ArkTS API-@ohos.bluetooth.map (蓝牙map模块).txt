# [@ohos.bluetooth.map (蓝牙map模块)](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-bluetooth-map)

## 目录
- [导入模块](#导入模块)
- [BaseProfile](#baseprofile)
- [map.createMapMseProfile](#mapcreatemapmseprofile)
- [MapMseProfile](#mapmseprofile)

本模块提供基于消息访问协议（Message Access Profile，[MAP](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/terminology#map)）的蓝牙消息访问能力，支持获取连接状态等方法。

 说明 本模块首批接口从API version 11开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

 
## 导入模块
 PhonePC/2in1TabletTVWearable收起自动换行深色代码主题复制
```
import { map } from '@kit.ConnectivityKit';
```

## map.createMapMseProfile
 PhonePC/2in1TabletTVWearablecreateMapMseProfile(): MapMseProfile

 创建蓝牙消息访问协议中的[MSE](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/terminology#mse)实例。通过该实例可使用本端作为MSE设备的接口，如：获取和其他设备间的蓝牙消息服务连接状态。

 **系统能力**：SystemCapability.Communication.Bluetooth.Core

 **返回值：**

 展开
| |
| |
| |
| MapMseProfile | 返回该profile的实例。 |

 以下错误码的详细介绍请参见[通用错误码说明文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/errorcode-universal)。

 展开
| |
| |
| |

| 801 | Capability not supported. |

 **示例：**

 收起自动换行深色代码主题复制
```
import { BusinessError } from '@kit.BasicServicesKit';
try { let mapMseProfile = map.createMapMseProfile(); console.info('MapMse success');} catch (err) { console.error('errCode: ' + (err as BusinessError).code + ', errMessage: ' + (err as BusinessError).message);}
```
 
## MapMseProfile
 PhonePC/2in1TabletTVWearable该实例表示蓝牙消息访问协议中的[MSE](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/terminology#mse)角色。

 - 该类继承于[BaseProfile](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-bluetooth-map#baseprofile)，因此可以使用其父类中的方法。
 - 使用该类的接口前，需通过[createMapMseProfile](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-bluetooth-map#mapcreatemapmseprofile)接口构造该类的实例。
 - 和该实例角色相对应的是[MCE](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/terminology#mce)角色。
