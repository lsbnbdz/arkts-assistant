# [@ohos.net.sharing (网络共享管理)](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-net-sharing)

## 目录
- [导入模块](#导入模块)
- [NetHandle](#nethandle)

网络共享管理模块用于将设备网络连接共享给其他连接设备。

 说明 本模块首批接口从API version 9开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

 
## 导入模块
PhonePC/2in1TabletTVWearable收起自动换行深色代码主题复制
```
import { sharing } from '@kit.NetworkKit';
```
