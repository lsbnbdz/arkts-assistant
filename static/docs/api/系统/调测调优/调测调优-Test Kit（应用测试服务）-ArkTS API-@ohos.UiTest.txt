UiTest提供模拟UI操作的能力，供开发者在测试场景使用，主要支持如点击、双击、长按、滑动等UI操作能力。

 该模块提供以下功能：

 - [On9+](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-uitest#on9)：提供控件特征描述能力，用于控件筛选匹配查找。
 - [Component9+](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-uitest#component9)：代表UI界面上的指定控件，提供控件属性获取，控件点击，滑动查找，文本注入等能力。
 - [Driver9+](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-uitest#driver9)：入口类，提供控件匹配/查找，按键注入，坐标点击/滑动，截图等能力。
 - [UiWindow9+](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-uitest#uiwindow9)：入口类，提供窗口属性获取，窗口拖动、调整窗口大小等能力。
 - [By(deprecated)](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-uitest#bydeprecated)：提供控件特征描述能力，用于控件筛选匹配查找。从API version 8开始支持，从API version 9开始废弃，建议使用[On9+](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-uitest#on9)替代。
 - [UiComponent(deprecated)](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-uitest#uicomponentdeprecated)：代表UI界面上的指定控件，提供控件属性获取，控件点击，滑动查找，文本注入等能力。从API version 8开始支持，从API version 9开始废弃，建议使用[Component9+](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-uitest#component9)替代。
 - [UiDriver(deprecated)](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-uitest#uidriverdeprecated)：入口类，提供控件匹配/查找，按键注入，坐标点击/滑动，截图等能力。从API version 8开始支持，从API version 9开始废弃，建议使用[Driver9+](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-uitest#driver9)替代。
 
 说明 - 本模块首批接口从API version 8开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。
 - 本模块接口在[自动化测试脚本](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/uitest-guidelines#%E4%BD%BF%E7%94%A8arkts%E6%8E%A5%E5%8F%A3%E8%BF%9B%E8%A1%8Cui%E6%B5%8B%E8%AF%95)中使用。
 - 本模块接口不支持并发调用。
