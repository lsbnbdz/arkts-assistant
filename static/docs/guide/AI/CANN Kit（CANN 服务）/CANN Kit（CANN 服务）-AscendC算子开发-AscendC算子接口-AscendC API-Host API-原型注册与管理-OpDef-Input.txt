## 函数功能
注册算子输入，调用该接口后会返回一个OpParamDef结构，后续可通过该结构配置算子输入信息。

 
## 函数原型
收起自动换行深色代码主题复制
```
OpParamDef &Input(const char *name);
```
 
## 参数说明
 展开
| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| name | 输入 | 算子输入名称。 |

 
## 返回值
[OpParamDef](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-opparamdef)算子参数定义。
