## 函数功能
获取当前硬件平台AI Core中Vector核数。若AI Core的架构为Cube、Vector分离架构，返回AI Core上的Vector核数；非分离架构返回AI Core的核数。

 
## 函数原型
收起自动换行深色代码主题复制
```
uint32_t GetCoreNumAiv(void) const;
```
 
## 参数说明
无

 
## 返回值
针对Kirin9020系列处理器，Cube、Vector分离架构，返回AI Core上的Vector核数。

 
## 约束说明
无
