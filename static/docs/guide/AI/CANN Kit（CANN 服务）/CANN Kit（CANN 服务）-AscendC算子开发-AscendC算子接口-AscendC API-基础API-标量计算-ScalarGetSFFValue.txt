## 功能说明
获取一个uint64_t类型数字的二进制中第一个0或1出现的位置，如果没找到则返回-1。

 
## 函数原型
收起自动换行深色代码主题复制
```
template __aicore__ inline int64_t ScalarGetSFFValue(uint64_t valueIn)
```
 
## 参数说明
 **表1**参数说明展开
| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| valueIn | 输入 | 输入数据，数据类型是uint64_t。 |
| countValue | 输入 | 获取到第一个0或1的位置。数据类型是int，值为0或1。 |

 
## 返回值
int64_t类型的数，valueIn中第一个0或1出现的位置。

 
## 支持的型号
Kirin9020系列处理器

 KirinX90系列处理器

 
## 约束说明
无。

 
## 调用示例
收起自动换行深色代码主题复制
```
uint64_t valueIn = 28; // 输出数据(oneCount): 2 int64_t oneCount = AscendC::ScalarGetSFFValue<1>(valueIn);
```
