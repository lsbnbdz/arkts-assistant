## 函数功能
当软件产生异常后，使用该指令使kernel中止运行。

 
## 函数原型
收起自动换行深色代码主题复制
```
__aicore__ inline void Trap()
```
 
## 参数说明
无

 
## 返回值
无

 
## 支持的型号

## 注意事项
该接口在kernel需要调试时使用。
