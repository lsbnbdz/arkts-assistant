## 功能说明
单核内Matmul矩阵相乘计算结束后必须调用一次End函数。

 
## 函数原型
收起自动换行深色代码主题复制
```
__aicore__ inline void End()
```
 
## 参数说明
无

 
## 返回值
无

 
## 支持的型号
