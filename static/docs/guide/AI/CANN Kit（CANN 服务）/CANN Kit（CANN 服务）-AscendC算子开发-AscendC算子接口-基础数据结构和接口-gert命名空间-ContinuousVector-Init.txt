## 函数功能
使用最大容量初始化本实例。

 
## 函数原型
收起自动换行深色代码主题复制
```
void Init(size_t capacity)
```
 
## 参数说明
 展开
| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| capacity | 输入 | 初始化本实例的容量。 |

 
## 返回值
无

 
## 约束说明
无

 
## 调用示例
收起自动换行深色代码主题复制
```
size_t capacity = 100U; size_t total_size = capacity * sizeof(int64_t) + sizeof(ContinuousVector); auto holder = std::unique_ptr(new (std::nothrow) uint8_t[total_size]); reinterpret_cast(holder.get())->Init(capacity); // 100U
```
