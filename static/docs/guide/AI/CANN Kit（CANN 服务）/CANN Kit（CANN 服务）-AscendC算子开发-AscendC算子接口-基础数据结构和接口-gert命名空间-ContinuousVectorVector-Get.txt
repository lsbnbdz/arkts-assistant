## 函数功能
获取第index个元素的首地址。

 
## 函数原型
收起自动换行深色代码主题复制
```
const ContinuousVector *Get(const size_t index) const
```
 
## 参数说明
 展开
| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| index | 输入 | 元素index。 |

 
## 返回值
第index个元素的首地址。

 
## 约束说明
无
