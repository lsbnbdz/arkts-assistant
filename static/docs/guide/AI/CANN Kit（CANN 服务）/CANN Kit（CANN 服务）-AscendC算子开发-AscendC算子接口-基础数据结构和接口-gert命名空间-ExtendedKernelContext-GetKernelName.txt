## 函数功能
获取当前内核的名称。

 
## 函数原型
收起自动换行深色代码主题复制
```
const char *GetKernelName() const
```
 
## 参数说明
无

 
## 返回值
当前内核的名称。

 
## 约束说明
无

 
## 调用示例
收起自动换行深色代码主题复制
```
// 假设已存在KernelContext *context auto extend_context = reinterpret_cast(context); auto kernel_name = extend_context->GetKernelName();
```
