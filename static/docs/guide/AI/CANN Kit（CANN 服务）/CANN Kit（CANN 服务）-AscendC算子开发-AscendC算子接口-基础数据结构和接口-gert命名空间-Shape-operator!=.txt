## 函数功能
判断与另一个Shape对象是否不等。

 
## 函数原型
收起自动换行深色代码主题复制
```
bool operator!=(const Shape &rht) const
```
 
## 参数说明
 展开
| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| rht | 输入 | 另一个Shape对象。 |

 
## 返回值
true：不相等。

 false：相等。

 
## 约束说明
无

 
## 调用示例
收起自动换行深色代码主题复制
```
Shape shape0({3, 256, 256}); Shape shape1({1, 3, 256, 256}); auto is_diff_shape = shape0 != shape1; // 返回值为true，不相等
```
