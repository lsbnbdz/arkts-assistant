## 函数原型
收起自动换行深色代码主题复制

```
const Shape &GetOriginShape() const
```

## 返回值

## 约束说明

## 调用示例
收起自动换行深色代码主题复制

```
StorageShape shape({3, 256, 256}, {256, 256, 3}); auto origin_shape = shape.GetOriginShape(); // 3,256,256
```
