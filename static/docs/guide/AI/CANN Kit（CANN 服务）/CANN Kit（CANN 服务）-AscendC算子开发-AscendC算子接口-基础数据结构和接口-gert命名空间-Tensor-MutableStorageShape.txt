## 函数功能
获取运行时Tensor的shape，此shape对象是可变的。

 
## 函数原型
收起自动换行深色代码主题复制
```
Shape &MutableStorageShape()
```
 
## 参数说明
无

 
## 返回值
运行时shape的引用。

 
## 约束说明
无

 
## 调用示例
收起自动换行深色代码主题复制
```
StorageShape sh({1, 2, 3}, {2, 1, 3}); Tensor t = {sh, {}, {}, ge::DT_FLOAT, nullptr}; auto shape = t.MutableStorageShape(); // 2,1,3
```
