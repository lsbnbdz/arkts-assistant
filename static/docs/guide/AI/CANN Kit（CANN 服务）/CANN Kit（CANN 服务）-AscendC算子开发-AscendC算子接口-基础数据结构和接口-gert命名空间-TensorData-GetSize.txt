## 函数功能
获取tensor数据的内存大小。

 
## 函数原型
收起自动换行深色代码主题复制
```
size_t GetSize() const
```
 
## 参数说明
无

 
## 返回值
tensor所占内存大小，单位为字节。

 
## 约束说明
无

 
## 调用示例
收起自动换行深色代码主题复制
```
auto addr = reinterpret_cast(0x10); TensorData td(addr, HostAddrManager, 100U, kOnHost); auto td_size = td.GetSize(); // 100U
```
