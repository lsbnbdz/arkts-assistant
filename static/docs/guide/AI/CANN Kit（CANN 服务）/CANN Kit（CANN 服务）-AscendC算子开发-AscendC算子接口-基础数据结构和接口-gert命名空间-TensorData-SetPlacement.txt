## 函数功能
设置tensor的placement。

 
## 函数原型
收起自动换行深色代码主题复制
```
void SetPlacement(const TensorPlacement placement)
```
 
## 参数说明
 展开
| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| placement | 输入 | tensor的placement。 关于TensorPlacement类型的定义，请参见TensorPlacement。 |

 
## 返回值
无

 
## 约束说明
无

 
## 调用示例
收起自动换行深色代码主题复制
```
auto addr = reinterpret_cast(0x10); TensorData td(addr, nullptr); auto td_place = td.SetPlacement(kOnHost);
```
