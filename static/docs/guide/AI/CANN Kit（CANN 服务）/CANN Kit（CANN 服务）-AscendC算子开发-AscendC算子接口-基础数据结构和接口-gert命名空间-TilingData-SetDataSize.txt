## 函数功能
设置tiling data长度。

 
## 函数原型
收起自动换行深色代码主题复制
```
void SetDataSize(const size_t size);
```
 
## 参数说明
 展开
| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| size | 输入 | tiling data长度。 |

 
## 返回值
无

 
## 约束说明
无

 
## 调用示例
收起自动换行深色代码主题复制
```
auto td_buf = TilingData::CreateCap(100U); auto td = reinterpret_cast(td_buf.get()); size_t data_size = td->GetDataSize(); // 0 td->SetDataSize(100U); data_size = td->GetDataSize(); // 100
```
