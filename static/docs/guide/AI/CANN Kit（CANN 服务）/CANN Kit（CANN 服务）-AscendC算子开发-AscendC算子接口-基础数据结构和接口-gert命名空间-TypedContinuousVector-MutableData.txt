## 函数功能
获取首个元素的指针地址，[GetData(), reinterpret_cast<T *>(GetData()) + GetSize())]中的数据即为当前容器中保存的数据。

 
## 函数原型
收起自动换行深色代码主题复制
```
T *MutableData()
```
 
## 参数说明
无

 
## 返回值
首个元素的指针地址。

 
## 约束说明
无

 
## 调用示例
收起自动换行深色代码主题复制
```
size_t capacity = 100U; auto cv_holder = ContinuousVector::Create(capacity); auto cv = reinterpret_cast(cv_holder.get()); auto cap = cv->MutableData(); 
```
