## 函数功能
将传入的DT类型（支持int64_t、float、std::string类型）的参数转换为对应T类型（支持INT、FLOAT、STR类型）的参数。

 - 支持将int64_t类型转换为INT类型。
- 支持将float类型转换为FLOAT类型。
- 支持将std::string类型转换为STR类型。

 
## 函数原型
收起自动换行深色代码主题复制
```
templatestatic T CreateFrom(DT &&val)
```
