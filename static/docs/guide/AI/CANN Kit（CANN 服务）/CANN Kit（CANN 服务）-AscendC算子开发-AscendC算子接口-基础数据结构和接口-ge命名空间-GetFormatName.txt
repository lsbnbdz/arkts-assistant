## 函数功能
根据传入的format类型，获取format的字符串描述。

 
## 函数原型
收起自动换行深色代码主题复制
```
const char_t *GetFormatName(Format format)
```
 
## 参数说明
 展开
| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| format | 输入 | format枚举值。 |

 
## 返回值
该format所对应的字符串描述，若format不合法或不被识别，则返回nullptr。

 
## 异常处理
无

 
## 约束说明
返回的字符串不可被修改。
