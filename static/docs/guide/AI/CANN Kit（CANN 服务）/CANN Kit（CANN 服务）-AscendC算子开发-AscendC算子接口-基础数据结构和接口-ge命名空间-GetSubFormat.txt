## 函数功能
从实际format中解析出子format信息。

 
## 函数原型
收起自动换行深色代码主题复制
```
inline int32_t GetSubFormat(int32_t format)
```
 
## 参数说明
 展开
| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| format | 输入 | 实际format（4字节大小，第1个字节的高四位为预留字段，低四位为c0 format段，第2-3字节为子format信息，第4字节为主format信息）。 |

 
## 返回值
实际format中包含的子format。
