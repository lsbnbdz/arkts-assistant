## 函数功能
根据子图名称获取算子对应的子图构建的函数对象。

 
## 函数原型
注意数据类型为string的接口后续版本会废弃，建议使用数据类型为非string的接口。

 收起自动换行深色代码主题复制
```
SubgraphBuilder GetSubgraphBuilder(const std::string &name) const;SubgraphBuilder GetSubgraphBuilder(const char_t *name) const;
```
