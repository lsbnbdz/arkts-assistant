## 函数功能
获取一个算子的子图名称列表。

 
## 函数原型
注意数据类型为string的接口后续版本会废弃，建议使用数据类型为非string的接口。

 收起自动换行深色代码主题复制
```
std::vector GetSubgraphNames() const;graphStatus GetSubgraphNames(std::vector &names) const;
```
