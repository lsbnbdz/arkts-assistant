## 函数功能
子图注册。

 
## 函数原型
收起自动换行深色代码主题复制
```
void SubgraphCountRegister(const char_t *ir_name, uint32_t count);
```
 
## 参数说明
 展开
| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| ir_name | 输入 | 子图名称。 |
| count | 输入 | 动态个数子图场景（子图数量不固定），注册count个数的子图，子图名称是ir_name_0 ~ ir_name_n， n < count。 |

 
## 返回值
无
