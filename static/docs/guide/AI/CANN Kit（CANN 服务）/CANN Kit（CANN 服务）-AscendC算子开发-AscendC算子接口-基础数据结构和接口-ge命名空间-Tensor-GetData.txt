## 函数功能
获取Tensor中的数据。

 const uint8_t* GetData() const返回的数据不可修改，uint8_t* GetData()返回的数据可修改。

 
## 函数原型
收起自动换行深色代码主题复制
```
const uint8_t *GetData() const;uint8_t *GetData();
```
