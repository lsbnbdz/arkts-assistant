## 函数功能
获取Tensor的补维规则。

 
## 函数原型
收起自动换行深色代码主题复制
```
graphStatus GetExpandDimsRule(AscendString &expand_dims_rule) const;
```
 
## 参数说明
 展开
| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| expand_dims_rule | 输入 | 函数待返回的expand_dims_rule补维规则，采用字符串形式表示补维。 |

 
## 返回值
 展开
| 类型 | 描述 |
| --- | --- |
| graphStatus | 设置成功返回GRAPH_SUCCESS，否则，返回GRAPH_FAILED。 |
