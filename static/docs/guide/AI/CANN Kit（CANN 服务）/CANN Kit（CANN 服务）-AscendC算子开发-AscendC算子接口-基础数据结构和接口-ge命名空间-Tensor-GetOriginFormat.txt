## 函数功能
获取Tensor的原始Format。

 该Format是指原始网络模型的Format。

 
## 函数原型
收起自动换行深色代码主题复制
```
ge::Format GetOriginFormat() const;
```
 
## 参数说明
无

 
## 返回值
 展开
| 类型 | 描述 |
| --- | --- |
| ge::Format | 返回tensor的原始Format值，默认值为FORMAT_RESERVED。 |
