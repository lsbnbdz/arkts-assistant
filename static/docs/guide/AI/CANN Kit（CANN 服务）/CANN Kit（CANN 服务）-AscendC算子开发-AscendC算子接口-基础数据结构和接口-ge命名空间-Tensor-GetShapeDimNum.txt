## 函数功能
获取shape的维度大小，即rank大小。

 
## 函数原型
收起自动换行深色代码主题复制
```
size_t GetShapeDimNum() const;
```
 
## 参数说明
无

 
## 返回值
 展开
| 类型 | 描述 |
| --- | --- |
| size_t | 返回shape的维度大小，即shape的rank，如果是unknown的rank，返回0。 |
