## 函数功能
获取Tensor的描述符。

 
## 函数原型
收起自动换行深色代码主题复制
```
TensorDesc GetTensorDesc() const;
```
 
## 参数说明
无

 
## 返回值
 展开
| 类型 | 描述 |
| --- | --- |
| TensorDesc | 返回当前Tensor的描述符。 |

 
## 异常处理
无

 
## 约束说明
修改返回的TensorDesc信息，不影响Tensor对象中已有的TensorDesc信息。
