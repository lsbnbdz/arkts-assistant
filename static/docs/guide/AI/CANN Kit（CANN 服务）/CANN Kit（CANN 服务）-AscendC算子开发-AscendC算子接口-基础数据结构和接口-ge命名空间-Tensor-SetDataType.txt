## 函数功能
设置Tensor的Datatype。

 
## 函数原型
收起自动换行深色代码主题复制
```
graphStatus SetDataType(const ge::DataType &dtype);
```
 
## 参数说明
 展开
| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| dtype | 输入 | 需设置的DataType值。 关于DataType类型，请参见DataType。 |

 
## 返回值
 展开
| 类型 | 描述 |
| --- | --- |
| graphStatus | 设置成功返回GRAPH_SUCCESS，否则，返回GRAPH_FAILED。 |
