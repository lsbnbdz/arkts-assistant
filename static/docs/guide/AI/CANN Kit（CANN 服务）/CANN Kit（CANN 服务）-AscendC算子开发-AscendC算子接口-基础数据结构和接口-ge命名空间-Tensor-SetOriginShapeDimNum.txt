## 函数功能
设置原始shape的维度大小，即rank大小。

 
## 函数原型
收起自动换行深色代码主题复制
```
graphStatus SetOriginShapeDimNum(const size_t dim_num);
```
 
## 参数说明
 展开
| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| dim_num | 输入 | 原始shape的维度大小，即原始shape的rank。 |

 
## 返回值
 展开
| 类型 | 描述 |
| --- | --- |
| graphStatus | 设置成功返回GRAPH_SUCCESS，否则，返回GRAPH_FAILED。 |
