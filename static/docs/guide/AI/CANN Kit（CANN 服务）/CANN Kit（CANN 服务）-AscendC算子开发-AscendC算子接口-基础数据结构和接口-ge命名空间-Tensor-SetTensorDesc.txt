## 函数功能
设置Tensor的描述符（TensorDesc）。

 
## 函数原型
收起自动换行深色代码主题复制
```
graphStatus SetTensorDesc(const TensorDesc &tensor_desc);
```
 
## 参数说明
 展开
| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| tensor_desc | 输入 | 需设置的Tensor描述符。 |

 
## 返回值
 展开
| 类型 | 描述 |
| --- | --- |
| graphStatus | 设置成功返回GRAPH_SUCCESS，否则，返回GRAPH_FAILED。 |
