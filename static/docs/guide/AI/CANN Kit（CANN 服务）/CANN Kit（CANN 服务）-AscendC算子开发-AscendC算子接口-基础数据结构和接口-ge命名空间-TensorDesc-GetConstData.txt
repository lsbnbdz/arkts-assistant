## 函数功能
如果TensorDesc是常量节点的描述，获取TensorDesc中的权重值。

 
## 函数原型
收起自动换行深色代码主题复制
```
bool GetConstData(uint8_t **const_data_buffer, size_t &const_data_len) const;
```
 
## 参数说明
 展开
| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| const_data_buffer | 输出 | 权重地址。 |
| const_data_len | 输出 | 权重长度。 |

 
## 返回值
获取成功，返回true。

 获取失败，返回false。

无
