## 函数功能
获取Tensor的补维规则。

 
## 函数原型
收起自动换行深色代码主题复制
```
graphStatus GetExpandDimsRule(AscendString &expand_dims_rule) const;
```
 
## 参数说明
 展开
| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| expand_dims_rule | 引用输入 | 获取到的补维规则，作为出参。 |

 
## 返回值
 展开
| 类型 | 描述 |
| --- | --- |
| graphStatus | 获取成功返回GRAPH_SUCCESS，否则，返回GRAPH_FAILED。 |
