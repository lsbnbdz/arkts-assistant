## 函数功能
获取TensorDesc所描述Tensor的原始Format。

 该Format是指原始网络模型的Format。

 
## 函数原型
收起自动换行深色代码主题复制
```
Format GetOriginFormat() const;
```
 
## 参数说明
无

 
## 返回值
 展开
| 类型 | 描述 |
| --- | --- |
| Format | TensorDesc所描述的Tensor的originFormat信息。 关于Format数据类型的定义，请参见Format。 |
