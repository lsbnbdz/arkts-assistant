## 函数功能
如果TensorDesc是常量节点的描述，向TensorDesc中设置权重值。

 
## 函数原型
收起自动换行深色代码主题复制
```
void SetConstData(std::unique_ptr const_data_buffer, const size_t &const_data_len);
```
 
## 参数说明
 展开
| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| const_data_buffer | 输入 | 权重地址。 |
| const_data_len | 输入 | 权重长度。 |
