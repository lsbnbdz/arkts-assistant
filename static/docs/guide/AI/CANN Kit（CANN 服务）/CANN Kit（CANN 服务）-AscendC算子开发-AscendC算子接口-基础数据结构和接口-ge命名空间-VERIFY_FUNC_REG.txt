## 函数功能
注册算子的Verify函数。

 
## 函数原型
收起自动换行深色代码主题复制
```
VERIFY_FUNC_REG(op_name, x)
```
 
## 约束说明
无

 
## 参数说明
 展开
| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| op_name | 输入 | 算子类型。 |
| x | 输入 | Verify函数名，和IMPLEMT_VERIFIER的Verify函数名保持一致。 |

 
## 返回值
无
