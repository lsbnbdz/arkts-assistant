## 场景介绍
 Enterprise Space Kit为应用提供查询工作空间信息的能力。

 
## 接口说明
 详细接口说明可参考[接口文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacemanager#section6592656143118)。

 展开
| 接口名 | 描述 |
| --- | --- |
| queryWorkspace(queryFlag: QueryType): Promise | 查询工作空间信息并返回结果。使用Promise异步回调。 |

 
## 开发步骤
 1. 导入Enterprise Space Kit模块。 收起自动换行深色代码主题复制 ``` import { spaceManager } from '@kit.EnterpriseSpaceKit'; ```
 2. 调用[queryWorkspace](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacemanager#section6592656143118)接口，查询工作空间，并且查看打印信息。 收起自动换行深色代码主题复制 ``` const queryFlag: spaceManager.QueryType = 0;try { const spaces: spaceManager.WorkspaceInfo[] = await spaceManager.queryWorkspace(queryFlag); console.info(`Succeeded in querying workspace` + JSON.stringify(spaces));} catch (err) { console.error(`Failed to query workspace. Code: ${err.code}, message: ${err.message}`);} ```
